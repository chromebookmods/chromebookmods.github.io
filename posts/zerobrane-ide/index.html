You may like to code in Lua-lang. However, you might not know how to install the IDE for that amazing coding language. Here we are back at another tutorial. This time it is a video tutorial.


First install the script using this code for the terminal. wget https://www.chromebookmods.ml/scripting/zerobrane-lua-ide-downloader.sh


Next use the command chmod u+x zerobrane-lua-ide-downloader-sh to make it an executable and then use the command ./zerobrane-lua-ide-downloader.sh to run it.

Now type "y" and press enter and it will load up. In the screenshot I wrote "zbstudio" because I forgot to update the code, but it's updated now so you don't have to type "zbstudio" once you run the script the first time. But if you want to run it after you installed it, you can run "zbstudio".

There you go, that's how you can use the IDE.

This tutorial was recorded with the Live OS method. You can check it out in one of the posts that has been made.
