How To Dual Boot on a Chromebook

To enable the compatibility for dual booting, you must make sure you have enabled developer mode. You can check it on one of our latest posts or click here. But if you have already turned it on, continue reading.


First press Ctrl+Alt+T on your chromebook. This will open up a crosh terminal like mine. Then type in "shell" and press enter.


Once that shows, type in cd; curl -LO https://mrchromebox.tech/firmware-util.sh on the input. What this will do is change the directory to a home directory (a safer place to put files) and then it will install a firmware script made by mrchromebox. You're screen should look similar to this.

Now run sudo install -Dt /usr/local/bin -m 755 firmware-util.sh. The command is very simple. It will install the file into the bin folder and also make it an executable.

Now we run the file with the command sudo firmware-util.sh. We use sudo so it has all privilleges to run it.

After it loads some yellow text, you will have a terminal UI (user interface). Type 1 and then press enter.

It will then say "default booting from USB?", type y and then press enter.

You can now press the enter key and press q to exit the program. Once that is completed, press Ctrl+Alt+Forward Key. This "forward key", by itself, allows you to navigate your back and forward pages, but use the contral and alt key this time. This will show you a black terminal. Type in chronos and then press enter.

Now as you cannot copy or paste in chronos, take a picture (or memorise) this command, sudo crossystem dev_boot_usb=1 dev_boot_legacy=1 and run it. Nothing will happen, when it's done (you'll see this when another "chronos@localhost" shows up) press Ctrl+Alt+Back Arrow to revert back.

Now, insert your USB and reboot your Chromebook. Then, before you press Ctrl+D, instead press Ctrl+L. Then press 1 (make sure your USB with the OS is the only one connected to the port, nothing else otherwise if you have another USB, you won't know which one is corresponds to a specific number for the boot).

Now you are done. If you want to revert back, reboot the Chromebook by pressing Power Button + Refresh and then press Ctrl+D. You can also remove your USB once you are back into ChromeOS. Also, if you are running Ubuntu, and there's an input shell, type help and press the "enter" key twice, then hopefully it should load (you may experience the screen glitching or rotating, I will cover a post on that soon). Another information, it might not say "booting from USB", if it does, your OS may not work (the one on the USB) so try another OS. If you want to run Windows (for Windows 7, you need an Intel Gen 5 or under) or Mac, it may not work. If it shows "2 - Tianocore Bootloader", it means you need to reinstall the RW_Legacy, so do these steps again. If you'd like to see more helpful images, click here. Apart from that, if you have any questions, go to the About page and click on TheSonicMaster's Discord as he can help you in the support channel. Thanks for reading and take care.
