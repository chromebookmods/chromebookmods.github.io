Let's say you want to compress a video, either to upload it somewhere or for your recent project. Compressing a video using FFMPEG is no hard task, quite easy if I do say so myself. There will be text and video formatted tutorials so you can choose.


First, install FFMPEG. If you are using Arch Linux do sudo pacman -S ffmpeg otherwise for Debian users, do sudo apt-get install FFMPEG


Next, run the command ffmpeg -i FILE_NAME_HERE -vcodec libx265 -crf 10 output.mp4. Before you run this, make sure to change a few things. Change "FILE_NAME_HERE" to your video file's name, and if you want to change the amount of size the file will be you can decrease the -crf 10 or increase it if you want. The higher it is, the bigger the file and vice versa. Also amke to change output.mp4 to whatever the file extension is. If you are changing a MKV file (for example) then do "output.mkv". You can also change the file name "output" to whatever you want.

That's it for this tutorial. There will be more FFMPEG tutorials on the recent posts (you should go check out "Other Mods").
