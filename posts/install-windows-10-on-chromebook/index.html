How To Install Windows 10 on a Chromebook

Let's see. ChromeOS is a bit of a dandy Operating System, especially if you were very used to Windows 10. Let's turn your dreams into reality and make your Chromebook into a Windows machine. Before you do so, special attributes towards MrChromebox and Coolstar.

We also need to make sure you are compatible so...

- [You will need an Intel Processor] Open up crosh with Ctrl+Alt+T and type uname -m. If it returns with x86_64, congratulations, you're compatible. Otherwise, sorry for the bummer (but you can always dual boot into Linux if you prefer that).
- You will also need 32GB or more storage. Windows itself is a behemoth of an operating system, taking up 20GB of storage with additional apps and constant updates. If you have a 16GB Chromebook, sorry you can't do it.
Make sure to check that your Chromebook is also listed on Coolstar because they have the drivers for that Chromebook. If yours is not listed there, you can't run Windows 10 due to the drivers.

Heads up

- Doing this will void your warranty, so you won't be able to ask for repairs.

- You will also lose all your Chrome OS files, data and applications that are not backed up. It is highly recommended you use Google's free 100GB Google Drive storage for Chromebook owners if you have a lot of files to back up.

- A small screwdriver (to remove a screw)

- Another Windows computer (to create Windows installation media).

- A USB flash drive of 8GB or larger (for Windows installation media).

- A USB mouse during Windows installation - the built-in trackpad will not work (although we will install a driver for it afterwards).



Create Windows 10 Installation File

Download Microsoft's Media Creation Tool. Save it to your other computer and run it with the USB drive inserted.



Accept the terms of service.

Choose the option "Create installation media".

Select your desired language; the edition should be "Windows 10" and architecture should be "64-bit (x64)" (32-bit is unsupported by the firmware).

Choose the "USB flash drive" option, then select your desired drive from the list that appears. For example, mine would be "E: (ESD-USB)".

Wait while the Windows installation media is being made. It can take up to 30 minutes.

When it is done remove the USB from your computer, it is now ready. Now we must remove the write protection screw.

Ensure your Chromebook is completely backed up to your satisfaction.
Turn off your Chromebook completely, until all lights are off (disconnect the charger if it is connected).
Flip the Chromebook over, remove the screws with your screwdriver (these screws are sometimes star shaped, it can be sold in packets) until the back cover comes off. Some Chromebooks' back plate pops off easily, others you may need to use some degree of force to pry it off.
Locate the write-protect screw and remove it. You may need to do some research to find it's location. For example, on the Acer Chromebook 14 CB3-431, it is located here:

Place the back panel back on, and put the screws back in. Ensure the entire panel is clicked in correctly before proceeding.

Now you must enable developer mode. Click here to see the tutorial.



Setup and Flash Firmware

After agreeing to the terms and conditions and connecting to your wifi, skip the sign in and click on "Browse as Guest".

Open up Chrome and press Ctrl+Alt+T to open crosh. Just like in the dual boot post, type "shell" and then type cd; curl -LO https://mrchromebox.tech/firmware-util.sh && sudo install -Dt /usr/local/bin -m 755 firmware-util.sh Once you run that, type sudo firmware-util.sh. A menu will pop up and show the following. Press 3; if the option is greyed out, then your device is not supported or write-protect is not disabled.

Press Y on these 2 prompts.

If you have a spare USB stick, you may take a firmware backup, however it is usually not needed since if you ever want to return to stock Chrome OS, they provide stock shellballs for you. I entered "n" at this prompt.

Now it will begin downloading and flashing the firmware. As stated here, it can take up to 90 seconds so be patient and do not turn off your device.

Now you will be returned to the script's main menu. From here, choose "p" to power off your device.

Plug in your Windows bootable USB stick, then turn on your Chromebook and allow it to boot the drive (first boot may take a long time, be patient!). You'll also need to plug in your USB mouse now.

Now this is a regular Windows 10 installation and quite simple. The installation would take 15-30 minutes minimum. Also if it asks you for your product key, click on "I don't have a product key"; select "Custom: Install Windows Only". You'll then see that none of your partitions are big enough to install Windows on. This is because you need to delete all the Chrome OS partitions (including EFI system partition) first, then select to install Windows to the unallocated space. Then allow it to install. Windows will reboot automatically, so you don't need to do anything apart from sit patiently (again) while it "Gets ready" and "Starts services". Then there will be another set up, just follow that and select your region and then you're nearly done. Once you see your brand new Windows 10 environment, eject your USB and input your external mouse. You will need to install neccessary drivers.



Installing Drivers

This is necessary because lots of your hardware (including trackpad and sound) will not work yet.

1. Head back to the CoolStar website and find your device again.
2. You should see links for the drivers here. Save them all to your computer.
3. Now you can go ahead and run each of the installers. Once finished, all hardware should be functional without needing a reboot.



That's pretty much it and Windows should be fully functional on your computer now! If you run into any issues, feel free to join my Discord server on the about section. Hopefully this helped you unleash your Chromebook's full potential!
